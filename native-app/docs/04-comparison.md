# 보일러플레이트 비교 및 선택 가이드

## 비교표

| 항목 | tauri-react-router | tauri-react |
|---|---|---|
| **라우팅** | React Router v7 (3개 라우트) | 없음 (단일 컴포넌트) |
| **레이아웃** | 사이드바 + Outlet (멀티 페이지) | 헤더 + 툴바 + 콘텐츠 (단일 페이지) |
| **프론트엔드 빌드** | `react-router build` | `vite build` |
| **빌드 출력** | `build/client/` | `dist/` |
| **Vite 플러그인** | tailwindcss → reactRouter → tsconfigPaths | tailwindcss → react |
| **경로 별칭** | `~/*` → `./app/*` | `~/*` → `./src/*` |
| **파일 저장** | Save + Save As | Save만 |
| **설정 페이지** | 있음 (업데이트 확인, 경로 표시, 외부 링크) | 없음 |
| **미저장 표시** | 빨간 `*` 별표 | 없음 |
| **파일 닫기** | 있음 (X 버튼) | 없음 |
| **복잡도** | 중간 (멀티 파일, 라우팅) | 낮음 (단일 파일) |
| **용도** | 여러 화면이 필요한 도구 앱 | 단순한 유틸리티/에디터 |
| **확장성** | 라우트 추가로 쉽게 확장 | 컴포넌트 분리 필요 |

## 선택 가이드

```
데스크톱 앱 필요
    │
    ├── 여러 화면/섹션이 필요한가?
    │   ├── YES → tauri-react-router
    │   │         (사이드바 네비게이션, 설정 화면, 확장 용이)
    │   │
    │   └── NO → 단순한 도구/유틸리티인가?
    │       ├── YES → tauri-react
    │       │         (빠른 시작, 최소 구조, 단일 화면)
    │       │
    │       └── 아직 모름 → tauri-react-router
    │                       (나중에 화면 추가 가능)
```

### tauri-react-router를 선택할 때

- 파일 관리 + 설정 화면이 모두 필요한 앱
- 사이드바/탭 기반 네비게이션이 필요한 앱
- 화면이 3개 이상인 도구 앱
- 향후 기능 확장 계획이 있는 프로젝트

→ 상세: [02-tauri-react-router.md](./02-tauri-react-router.md)

### tauri-react를 선택할 때

- 단일 목적의 유틸리티 (예: 텍스트 변환기, 이미지 뷰어)
- 화면 전환이 필요 없는 앱
- 최소한의 코드로 빠르게 시작하고 싶을 때
- 프로토타입/MVP

→ 상세: [03-tauri-react.md](./03-tauri-react.md)
